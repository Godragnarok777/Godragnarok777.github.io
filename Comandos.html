Comandos:

Cow tiene un conjunto muy limitado de comandos, lo que lo hace muy fácil de aprender.

Instrucción Descripción
moo Este comando está conectado al comando MOO. Cuando se encuentra durante la ejecución normal, busca en el código del programa a la inversa un comando MOO coincidente y comienza a ejecutarse de nuevo a partir del comando MOO encontrado. Al realizar una búsqueda, se salta la instrucción que está inmediatamente antes de ella (ver MOO).
mOo Mueve la posición actual de la memoria hacia atrás un bloque.
moO Mueve la posición actual de la memoria hacia adelante un bloque.
mOO Ejecute el valor en el bloque de memoria actual como si fuera una instrucción. El comando ejecutado se basa en el valor del código de instrucción (por ejemplo, si el bloque de memoria actual contiene un 2, se ejecuta el comando moO). Un comando no válido sale del programa en ejecución. El valor 3 no es válido, ya que provocaría un bucle infinito.
Moo Si el bloque de memoria actual tiene un 0, lea un solo carácter ASCII de STDIN y guárdelo en el bloque de memoria actual. Si el bloque de memoria actual no es 0, imprima el carácter ASCII que corresponde al valor del bloque de memoria actual en STDOUT.
MOo Disminuya el valor actual del bloque de memoria en 1.
MoO Incremente el valor actual del bloque de memoria en 1.
MOO Si el valor actual del bloque de memoria es 0, omita el siguiente comando y reanude la ejecución después del siguiente comando moo coincidente. Si el valor actual del bloque de memoria no es 0, continúe con el siguiente comando. Tenga en cuenta que el hecho de que omita el comando inmediatamente posterior tiene ramificaciones interesantes para saber dónde está realmente el comando moo correspondiente. Por ejemplo, lo siguiente coincidirá con el segundo moo y no con el primero: OOO MOO moo moo
OOO Establezca el valor actual del bloque de memoria en 0.
MMM Si no hay ningún valor actual en el registro, copie el valor actual del bloque de memoria. Si hay un valor en el registro, pegue ese valor en el bloque de memoria actual y borre el registro.
OOM Imprime el valor del bloque de memoria actual en STDOUT como un número entero.
oom Lea un número entero de STDIN y colóquelo en el bloque de memoria actual.

En general, Cow es un lenguaje de programación muy sencillo y fácil de aprender, pero su limitado conjunto de comandos lo hace menos adecuado para programas complejos.
